== Description ==

**WooCommerce Sale Funnel** plugin enables the admin to create proper funnel channels onto their webpage in order to provide a channeled search option of the discounts and the ongoing offers-Place your funnel wisely.

Merchants can create funnels for launching a new product or creating a sale funnel that offers the ability to manage to order by adding or deleting funnel steps-Update your customers with the most discounted products.

WooCommerce Sale Funnel has come up with 10% conversion rate for the e-commerce websites as stated by a recent survey. 

It removes the dependency of the admin on developers who cost excessively- WooCommerce Sale Funnel offers a code-free solution to the vendors.

== Frequently Asked Questions ==

= Installation Instructions =

1. Make sure WooCommerce and Forge are installed previously.
2. Go to the Funnel Post Type and create a funnel.
3. You can choose between a Sell Funnel or new Product Launch.
4. We have provided default Steps you can use modify, delete or create new steps.
5. You have assigned one default forge template to each funnel step. 
6. Option to edit or change the template of any step.
7. On editing, you can see all the components we have provided you can drag and drop them to respected funnel steps.
